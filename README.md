# 🎮 Rock Paper Scissors

A simple and interactive **Rock Paper Scissors** game built using **HTML, CSS, and JavaScript**. The game allows users to play against the computer with real-time score updates and dynamic game messages.

---

## 🚀 Features

* Interactive gameplay
* Random computer move generation
* Live score tracking
* Win, lose, and draw detection
* Dynamic result messages
* Responsive and clean user interface

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## ▶️ How to Run

1. Clone this repository.
2. Open the project folder.
3. Open `index.html` in your browser.

No additional installation or dependencies are required.
